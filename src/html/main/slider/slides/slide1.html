Регистрация и добавление проекта в систему

  • 1. Зарегистрируйтесь в нашем личном кабинете
  • 2. При добавление проекта вам необходимо указать URL (адрес) своего сайта
  • 3. Для вашего сайта будет сгенерирован уникальный код, который необходимо установить на свой сайт (в шаблоне сайта, перед тегом )
